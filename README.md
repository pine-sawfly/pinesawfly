PineSawFly
==========

PySide6 + Qt Quick/QML 桌面安全审计工具。

运行：

```powershell
uv sync
uv run python -m pinesawfly
```

旧入口仍可用：

```powershell
uv run python main.py
```

资源约定：

- 应用图标：`app.ico`
- 字体：`assets/fonts/`
- Material Icons ligature 字体：`assets/fonts/MaterialIcons-Regular.ttf`
- 其他图标素材：`assets/icons/`

如果导航图标显示成 `home`、`settings` 这类文字，说明缺少 Material Icons 字体。下载或提供对应 TTF 后放到上面的路径即可。
